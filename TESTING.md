# testing in hexi

run tests:
```bash
cargo test --tests
```

run specific test:
```bash
cargo test --test assertions
cargo test --test hex_tests
cargo test --test speed_tests
```

run with output:
```bash
cargo test --tests -- --nocapture
```

## what have i done

- **assertions.rs**: fool_check logic and args creation speed
- **hex_tests.rs**: hex dump functionality and large data handling
- **speed_tests.rs**: performance of file I/O and hex output

all tests pass. takes about 1 second to run everything.
